朔間凜月(さくま りつ)


身高/體重 : 170cm/54kg
血型 : AB
生日 : 9/22
職業 : 偶像 (所屬團體 : Knights)
興趣 : 睡覺
特技 : 彈鋼琴

簡介 :
朔間家次男，因家族遺傳緣故而懼怕陽光，白天時沒什麼精神大多時間都在睡覺。 雖說個性比較隨興自我，但是在加速思考的時候說話非常的直接。
一開始非常排斥夕依的存在，但知道對方的身世以及被同樣被家族綁住的事實後就開始互相扶持，某次因為暴走而吸取夕依的血液後就對夕依產生了依賴性，後來認知到自己早已喜歡上夕依的事實。

佐久間 夕依(さくま ゆい)


身高/體重 : 156cm/保密
血型 : B
生日 : 6/10
職業 : 網頁工程師
興趣 : 旅行
特技 : 很容易哭(?)

簡介 :
是一位台日混血兒，在高中時被朔間家帶到日本進行領養，成為朔間家的一份子。
在20歲成年後歸國籍成台灣人(因日本成年後無法持有雙國籍)。

佐久間家以前為魔女的後裔並與朔間家交好，後來卻因被人獵巫導致家族滅亡，佐久間夕依繼承了魔女的能力成為了世上稀少的魔女後繼者。
不願意被家庭綁住的夕依在到日本後進到夢之咲的普通科就讀，學習跟營養有關的知識幫凜月調理身體，大學後學習程式與設計，成年後回到台灣成為一名網頁設計師。

繪者 : 老大
